Gestión de Infraestructura


  • La gestión se hace en varios niveles:

    • Containers: Proxmox VE

Proxmox VE


Proxmox VE es una solución open source que integra la gestión de máquinas virtuales KVM, containers LXC, almacenamiento definido por software redes a través de una interfaz de usuario centralizada.



Características de Proxmox


  • Open Source y gratuito (soporte pago).
  • Soporta KVM y LXC.
  • Administración centralizada: web, CLI y API REST.
  • Alta diponibilidad.
  • Otros servicios: migración en caliente, replicación, backups, redes, firewall, storage.



Alta Disponibilidad en Proxmox

  • Cluster Multi-master sin único punto de fallo.
  • Opciones de HA disponbles en la interfaz.
  • Sistema de archivos distribuido y replicado para la configuración del cluster y nodos.
  • Agentes de monitoreo para KVM y containers LXC.
  • Fencing basado en watchdogs (hw o sw).

Storage en Proxmox


  • Alm. local usando ZFS, LVM, LVMthin, ext4 y XFS.
  • Alm. remoto usando FC, iSCSI o NFS.
  • Alm. distribuido con Ceph RBD o CephFS.
  • Encriptación sobre Ceph OSD y ZFS.
  • Número ilimitado de definiciones de storage.


Otras Caracteríticas de Proxmox


  • Interfaz de línea de comandos
  • API REST
  • Redes virtualizadas
  • Fuentes de Autenticación y RBAC
  • Templates de VM y clones
  • Backup y restore manuales y programados.

Rancher


Rancher es una plataforma de administación de containers pensada para las organizaciones que despliegan containers en producción. Rancher permite correr kubernetes en cualquier lugar, cumplir con requisitos y regulaciones, y facilitar la labor de equipos DevOps.

  • Administración centralizada: web, CLI y API REST.
  • Alta diponibilidad.
